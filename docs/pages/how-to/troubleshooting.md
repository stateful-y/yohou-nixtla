# Troubleshooting

Solutions to common problems when using Yohou-Nixtla.

## Installation Issues

**Problem: Package not found**
: Verify the package name and that it is published: `pip install yohou_nixtla`

**Problem: Import error after installation**
: Make sure you installed in the active environment: `python -c "import yohou_nixtla"`

## Runtime Issues

**Problem: Unexpected behavior**
: Check your configuration and consult the [Configuration guide](configure.md).
