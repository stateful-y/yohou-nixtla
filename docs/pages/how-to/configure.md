# How to Configure Yohou-Nixtla

This guide shows you how to set up and customize Yohou-Nixtla for your project.

## Basic Configuration

```python
from yohou_nixtla import [MainClass]

instance = [MainClass](
    option_1="value",
    option_2=True,
)
```

## Configuration File

```yaml
# config.yaml
yohou_nixtla:
  option_1: value
  option_2: true
```
